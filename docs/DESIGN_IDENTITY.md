# სამეზობლოს — ვიზუალური იდენტობა (საკუთარი ინტერპრეტაცია)

> **არა Archi-ს კოპია.** ფუნქციები იგივეა (გადახდა, სერვისი, მარკეტინგი), ვიზუალი — თბილი, საზოგადოებრივი, „მეზობლობის“ აპი.

## კონცეფცია

| Archi (რეფერენსი) | სამეზობლოს (ჩვენი) |
|-------------------|---------------------|
| კორპორატიული ლურჯი | ბუნებრივი ტილი + თბილი ნარინჯი |
| მუქი ჰედერი | ღია გრადიენტი + მრგვალი ფორმები |
| 5 ტაბი (ქონება, პროექტი, ქარდი…) | 4 ტაბი — მარკეტინგი ცენტრში |
| „ხარვეზის დაფიქსირება“ | „დახმარება“ — მეგობრული ენა |
| ლურჯი წრე „უკან“ | მარტივი უკან ტექსტი |
| ვარდისფერი empty | ილუსტრაცია + რბილი ტექსტი |

## ტაბები (4)

0. **შესვლა** — მომხმარებელი + პაროლი (აპის გახსნამდე)  
1. **მთავარი** — დავალიანება, სწრაფი მოქმედებები, სიახლე  
2. **მარკეტინგი** — აქციები, შეთავაზებები, სიახლეები  
3. **დახმარება** — მოთხოვნები, ახალი ფორმა  
4. **ჩემი სივრცე** — ბინა, დოკუმენტები, პარამეტრები
